AADC background

  • established 1995: long-term management of Australia's Antarctic data
  • motivations: Treaty obligations and desire for good science
  • international and national engagement

Science scope

http://www.antarctica.gov.au/science

  • climate, sea ice, atmosphere, oceans
  • terrestrial, marine, and coastal ecosystems
  • fisheries, conservation, biodiversity
  • human impacts, change and protection
  • astronomy, geosciences, space weather, fundamental biology and physiology

Metadata

  • early on = ANZLIC standard, now = DIF

  • GCMD provides all metadata services for AADC

  • AADC retains metadata officer control of records and local copies of metadata records

    integration

    contribution to other GCMD portals

    conversion to other formats for harvesting

Metadata authoring

buy-in from users requires a friendly authoring tool

  • authoring process is broadly:

    • user creates/edits metadata record
    • checked by metadata officer
    • published

Authoring tools

  • ANZLIC/DIF tools not well regarded by average user: not intuitive, technical terminology

    • poor quality records that required a lot of editing

  • from 2012, simple Word template with 8 questions

    • much better uptake and quality of input
    • but effort to convert to DIF

Metadata authoring app

  • own metadata authoring tool

    • many fields removed or automatically set
    • produces DIF XML that requires minimal modification
    • integrated with AADC applications (e.g. EDS)
    • released October 2015, early testing is positive
